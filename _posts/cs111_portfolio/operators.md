# Operators

## Overview
This objective demonstrates proficiency in using different operators to perform computations, comparisons, and logical operations.

---

## Sub-Deliverables

### 1. Arithmetic Operators
- **Description**: Perform mathematical operations using arithmetic operators
- **Key Concepts**:
  - Addition (`+`)
  - Subtraction (`-`)
  - Multiplication (`*`)
  - Division (`/`)
  - Modulus (`%`)
  - Exponentiation (`**`)
  - Operator precedence
- **Example**: Calculate total cost, discount, and final price

### 2. Assignment Operators
- **Description**: Assign and update variable values
- **Key Concepts**:
  - Simple assignment (`=`)
  - Compound assignment (`+=`, `-=`, `*=`, `/=`, `%=`)
  - Increment (`++`) and decrement (`--`)
  - Pre vs. post increment
- **Example**: Update counters and accumulators

### 3. Comparison Operators
- **Description**: Compare values and return boolean results
- **Key Concepts**:
  - Equality (`==` vs. `===`)
  - Inequality (`!=` vs. `!==`)
  - Greater than (`>`)
  - Less than (`<`)
  - Greater than or equal (`>=`)
  - Less than or equal (`<=`)
- **Example**: Check if score meets passing threshold

### 4. Logical Operators
- **Description**: Combine conditions using logical operations
- **Key Concepts**:
  - AND (`&&`) operator
  - OR (`||`) operator
  - NOT (`!`) operator
  - Operator precedence
  - Short-circuit evaluation
- **Example**: Check multiple conditions in if statement

### 5. Bitwise Operators
- **Description**: Perform operations on binary representations
- **Key Concepts**:
  - AND (`&`)
  - OR (`|`)
  - XOR (`^`)
  - NOT (`~`)
  - Left shift (`<<`)
  - Right shift (`>>`)
- **Example**: Check flag status or bit manipulation

### 6. String Operators
- **Description**: Concatenate and manipulate strings
- **Key Concepts**:
  - Concatenation (`+`)
  - Template literals (`` ` ``)
  - String methods
  - Interpolation
- **Example**: Combine strings to create messages

### 7. Ternary Operator
- **Description**: Use inline conditional operator
- **Key Concepts**:
  - Ternary syntax (`condition ? true : false`)
  - Nested ternaries
  - When to use vs. if/else
  - Readability considerations
- **Example**: Assign value based on condition

### 8. Special Operators
- **Description**: Work with special operators for advanced operations
- **Key Concepts**:
  - typeof operator
  - instanceof operator
  - Spread operator (`...`)
  - Nullish coalescing (`??`)
  - Optional chaining (`?.`)
- **Example**: Check variable type or provide default values
