# Control Structures

## Overview
This objective demonstrates proficiency in using conditional statements and loops to control program flow and handle different scenarios.

---

## Sub-Deliverables

### 1. Conditional Statements (If/Else)
- **Description**: Use conditional logic to execute different code paths
- **Key Concepts**:
  - `if` statements
  - `else if` and `else` branches
  - Logical operators (`&&`, `||`, `!`)
  - Comparison operators (`==`, `!=`, `<`, `>`, `<=`, `>=`)
- **Example**: Check user age to determine eligibility

### 2. Switch Statements
- **Description**: Implement multi-branch selection using switch cases
- **Key Concepts**:
  - `switch` statement structure
  - `case` blocks
  - `break` statements
  - `default` case
- **Example**: Menu system with different actions based on user choice

### 3. Ternary Operators
- **Description**: Use inline conditional expressions for simple conditions
- **Key Concepts**:
  - Ternary operator syntax (`condition ? true : false`)
  - Nested ternaries
  - When to use vs. if/else
- **Example**: Assign pass/fail based on score

### 4. For Loops
- **Description**: Iterate a fixed number of times using for loops
- **Key Concepts**:
  - Loop initialization, condition, and increment
  - Loop counters
  - Loop control (break, continue)
  - Nested loops
- **Example**: Print multiplication table or process array elements

### 5. While Loops
- **Description**: Iterate based on a condition using while loops
- **Key Concepts**:
  - While loop structure
  - Loop condition evaluation
  - Infinite loop prevention
  - Loop control statements
- **Example**: Input validation loop or game loop

### 6. Do-While Loops
- **Description**: Execute code at least once before checking condition
- **Key Concepts**:
  - Do-while structure
  - Execute-then-check pattern
  - Menu-driven programs
- **Example**: Menu that runs at least once

### 7. Nested Control Structures
- **Description**: Combine multiple control structures together
- **Key Concepts**:
  - Loops within loops
  - Conditionals within loops
  - Complex logic patterns
  - Managing nested complexity
- **Example**: Search grid with nested loops and conditions

### 8. Loop Optimization and Patterns
- **Description**: Write efficient and readable loop patterns
- **Key Concepts**:
  - Early termination (break)
  - Skip iterations (continue)
  - Common loop patterns
  - Performance considerations
- **Example**: Finding first match in collection

---

## Evaluation Criteria
- [ ] Conditionals used appropriately for logic flow
- [ ] All loop types used where suitable
- [ ] Control statements (break, continue) applied correctly
- [ ] No infinite loops or logic errors
- [ ] Nested structures work as intended
- [ ] Code is readable and efficient
